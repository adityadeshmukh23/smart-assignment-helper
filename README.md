## Overview
Smart Assignment Helper accepts an assignment brief and performs: reasoning (extract objectives), planning (milestones), and execution (generate README, skeleton code, and retrieve resources via RAG). Interaction logs and prompts are saved.

## Run (MVP)
1. `pip install -r requirements.txt`
2. `export OPENAI_API_KEY="sk-..."`
3. `streamlit run frontend/app.py`
4. Use the UI to paste assignment and run Planner & Executor.
